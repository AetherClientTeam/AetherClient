# Aether Settings Mockups

These SVGs are the approval gate before the C++ accordion menu is implemented.
They show the same catalog at the required viewport and UI-scale combinations:

- `aether-settings-1920x1080-100.svg`
- `aether-settings-1920x1080-150.svg`
- `aether-settings-1280x720-100.svg`
- `aether-settings-1280x720-150.svg`

The mockups intentionally use generic independent feature names. They define
layout and interaction density, not final feature behavior or protected UI.
